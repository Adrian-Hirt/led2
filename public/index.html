About

This site was created for the Bastli LED Contest in spring 2016.

Technical

It uses Node.js, Angular.js and Socket.IO and of course standart Javascript to turn the mp3 into a visualisation using the LEDs in the Aufenthaltsraum.
This site is based on a visualizer I made in javascript, which can be found on my website.

It uses the AudioContext API built into modern browsers to analyze the song, then sends it via Socket.io to the Node.js backend. The backend then turns the data it got into RGB values for the LEDs, which are finally sent to the LED stripes via UDP.
